Os dados de modelos atmosféricos normalmente são gerados em formatos do tipo grib, grib2 ou netcdf.

Para gerar os campos de previsão de precipitação do modelo CFSv2 (Climate Forecast System) foi criado o script gera_mapa.ipynb.

Os dados podem ser baixados do site do NCEP, disponíveis em: https://www.nco.ncep.noaa.gov/pmb/products/cfs/
e os dados utilizados neste exemplo estão disponíveis em: https://drive.google.com/drive/folders/1ZKYwSO8kNkpEqStq_fKc3h8UrEbwBPkO?usp=sharing
